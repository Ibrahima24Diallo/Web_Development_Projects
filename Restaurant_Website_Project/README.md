# Restaurant Website Project

This project is a web application that simulates a restaurant website. It shows a menu, different food categories, and allows users to explore items in a simple and interactive way.

## Features
Responsive Design
- The website works well on desktop, tablet, and mobile devices using Bootstrap and CSS.
- Dynamic Content Loading
- Menu categories and items are loaded using JavaScript and Ajax without refreshing the page.
- Menu Navigation
- Users can easily browse different food categories and view menu items.
- Random Category Feature
- A “Special” button shows a random menu category each time it is clicked.

## Technologies Used
- HTML
- CSS
- JavaScript
- Ajax
- Bootstrap

This project helped me practice building responsive websites and using JavaScript to load data dynamically. It also improved my problem-solving and front-end development skills.
